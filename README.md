# Retrieval Augmented Generation (RAG) Project

## Overview
This project demonstrates the concept of **Retrieval Augmented Generation (RAG)** using Python.  
RAG combines information retrieval with large language models to generate accurate and context-aware responses.

## Features
- Retrieves information from a text dataset
- Uses LLM to generate responses
- Demonstrates the working principle of RAG systems

## Technologies Used
- Python
- LangChain
- LLM Integration
- Text Dataset

## Project Structure
Rag-project/
│
├── Rag.py
├── cat-facts.txt
└── README.md

## How It Works
1. The system loads a dataset (`cat-facts.txt`)
2. Relevant information is retrieved
3. The language model generates answers based on retrieved context
